# F14 可上傳 orbit icon + 圖庫 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 讓卡片「亮」（needs-attention）時沿框繞圈的純光點，可選擇疊一張上傳的 icon 騎在亮頭上一起繞（上大下小、貼亮頭、支援動圖），三狀態各可指派一張，未指派維持現況純光點。

**Architecture:** 完全複用 F13 的 `makeUploadStore`（開第二個 `subdir:'pic'` 實例），server 端鏡射 voice 的 `/api/pic/*` + `/uploads/pic/` 路由（差異只在 image MIME）。FE 把 F13 的「鈴聲設定」modal 升級為單一「提醒設定」modal、加一個「Orbit icon」區（鏡射 voice 圖庫 UI）。orbit icon 用真 `<img class="orbit-icon">` 放進卡片 template，CSS `offset-path` 沿圓角矩形周長繞，keyframe 時間對齊既有 `orbit-marquee`；可見性/動畫純由 CSS cascade（`.card.needs-attention .orbit-icon:not([hidden])`）驅動，沒指派時 conic 機制一個 byte 都不碰。

**Tech Stack:** Vanilla Node `http`（`server/index.js`）+ `server/lib/uploads.js`（既有泛型工廠，**不改**）；vanilla ES module 前端（`web/ui/app.js`）+ `web/dashboard.html` + `web/ui/styles.css`（CSS `offset-path` / `@keyframes`）；`node:test` 單元測試。全 repo 僅依賴 `ws`，**不得加任何套件**。

```js
const { test } = require('node:test');
const assert = require('node:assert');
const fs = require('fs');
const path = require('path');
const { makeUploadStore } = require('../lib/uploads');

const SUB = '__test_pic__';
const DIR = path.join(__dirname, '..', '..', 'uploads', SUB);
const PIC_EXT = ['png', 'jpg', 'jpeg', 'gif', 'webp', 'svg'];

function fresh() {
  fs.rmSync(DIR, { recursive: true, force: true });
  return makeUploadStore({
    subdir: SUB,
    allowExt: PIC_EXT,
    maxBytes: 2 * 1024 * 1024,
    events: ['waiting', 'completed', 'failed'],
  });
}
test.after(() => fs.rmSync(DIR, { recursive: true, force: true }));

test('save accepts every allowed image extension', () => {
  const s = fresh();
  for (const ext of PIC_EXT) {
    const r = s.save(`icon.${ext}`, Buffer.from('x'));
    assert.equal(r.name, `icon.${ext}`);
  }
  assert.equal(s.list().length, PIC_EXT.length);
});

test('save rejects a non-image extension', () => {
  const s = fresh();
  assert.throws(() => s.save('note.txt', Buffer.from('x')), /extension/);
});

test('save rejects a file over 2 MB', () => {
  const s = fresh();
  assert.throws(() => s.save('big.png', Buffer.alloc(2 * 1024 * 1024 + 1)), /too large/);
});

test('safeName rejects path traversal', () => {
  const s = fresh();
  assert.throws(() => s.safeName('../x.png'), /invalid name/);
  assert.throws(() => s.safeName('a/b.png'), /invalid name/);
  assert.throws(() => s.safeName('a\\b.png'), /invalid name/);
});

test('delete removes the file and clears its assignment', () => {
  const s = fresh();
  s.save('bell.gif', Buffer.from('abc'));
  s.assign('completed', 'bell.gif');
  s.remove('bell.gif');
  assert.equal(s.list().length, 0);
  assert.equal(s.readAssignments().completed, null);
});

test('assign handles the three states and null (= 光點)', () => {
  const s = fresh();
  s.save('a.png', Buffer.from('a'));
  assert.throws(() => s.assign('bogus', 'a.png'), /invalid event/);
  assert.throws(() => s.assign('waiting', 'missing.png'), /file not found/);
  assert.equal(s.assign('waiting', 'a.png').waiting, 'a.png');
  assert.equal(s.assign('completed', 'a.png').completed, 'a.png');
  assert.equal(s.assign('failed', 'a.png').failed, 'a.png');
  assert.equal(s.assign('waiting', null).waiting, null);
});

test('list ignores assignments.json', () => {
  const s = fresh();
  s.writeAssignments({ waiting: null });
  assert.equal(s.list().length, 0);
});
```

在 voice 的 `/uploads/voice/` 區塊（結束於 `:473` 的 `return;` 後）與 `// F12: shut down`（`:475`）之間，插入下列整段（結構鏡射 voice，把 `voice`→`pic`、`VOICE_MAX_BYTES`→`PIC_MAX_BYTES`、audio MIME→image MIME）：

```js
  // ============== F14: orbit icon library (pic) — mirrors voice block ==============
  if (req.method === 'GET' && url.pathname === '/api/pic/list') {
    res.writeHead(200, { 'Content-Type': 'application/json; charset=utf-8' });
    res.end(JSON.stringify({ files: picStore.list(), assignments: picStore.readAssignments() }));
    return;
  }

  if (req.method === 'POST' && url.pathname === '/api/pic/upload') {
    const name = url.searchParams.get('name') || '';
    const chunks = [];
    let size = 0, aborted = false;
    req.on('data', (c) => {
      size += c.length;
      if (size > PIC_MAX_BYTES) {
        aborted = true;
        res.writeHead(400, { 'Content-Type': 'application/json; charset=utf-8' });
        res.end(JSON.stringify({ error: 'file too large' }));
        req.destroy();
        return;
      }
      chunks.push(c);
    });
    req.on('end', () => {
      if (aborted) return;
      try {
        const file = picStore.save(name, Buffer.concat(chunks));
        res.writeHead(200, { 'Content-Type': 'application/json; charset=utf-8' });
        res.end(JSON.stringify({ ok: true, file }));
      } catch (e) {
        res.writeHead(400, { 'Content-Type': 'application/json; charset=utf-8' });
        res.end(JSON.stringify({ error: e.message }));
      }
    });
    return;
  }

  if (req.method === 'POST' && url.pathname === '/api/pic/delete') {
    let body = '';
    req.on('data', (c) => { body += c.toString(); });
    req.on('end', () => {
      try {
        const reqName = String(JSON.parse(body || '{}').name || '');
        picStore.remove(reqName); // also clears any assignment pointing at it
        res.writeHead(200, { 'Content-Type': 'application/json; charset=utf-8' });
        res.end(JSON.stringify({ ok: true, assignments: picStore.readAssignments() }));
      } catch (e) {
        res.writeHead(400, { 'Content-Type': 'application/json; charset=utf-8' });
        res.end(JSON.stringify({ error: e.message }));
      }
    });
    return;
  }

  if (req.method === 'POST' && url.pathname === '/api/pic/assign') {
    let body = '';
    req.on('data', (c) => { body += c.toString(); });
    req.on('end', () => {
      try {
        const b = JSON.parse(body || '{}');
        const event = String(b.event || '');
        const name = b.name == null ? null : String(b.name);
        const assignments = picStore.assign(event, name);
        res.writeHead(200, { 'Content-Type': 'application/json; charset=utf-8' });
        res.end(JSON.stringify({ ok: true, assignments }));
      } catch (e) {
        res.writeHead(400, { 'Content-Type': 'application/json; charset=utf-8' });
        res.end(JSON.stringify({ error: e.message }));
      }
    });
    return;
  }

  // Serve uploaded images for the orbit <img>. filePath() validates name + ext and
  // rejects path separators, neutralizing traversal (mirrors /uploads/voice/).
  if (req.method === 'GET' && url.pathname.startsWith('/uploads/pic/')) {
    const name = decodeURIComponent(url.pathname.slice('/uploads/pic/'.length));
    let fp;
    try { fp = picStore.filePath(name); }
    catch { res.writeHead(403); res.end('forbidden'); return; }
    try {
      const body = fs.readFileSync(fp);
      const ext = path.extname(fp).toLowerCase();
      const type = ext === '.png' ? 'image/png'
        : ext === '.jpg' || ext === '.jpeg' ? 'image/jpeg'
        : ext === '.gif' ? 'image/gif'
        : ext === '.webp' ? 'image/webp'
        : ext === '.svg' ? 'image/svg+xml' : 'application/octet-stream';
      res.writeHead(200, { 'Content-Type': type, 'Cache-Control': 'no-store' });
      res.end(body);
    } catch (err) {
      res.writeHead(404); res.end('image not found');
    }
    return;
  }
```

- [ ] **Step 1: 加 pic 圖庫 state + load/render/delete**

在 voice 圖庫區（`app.js:206-253`，即 `renderVoiceLib`/`deleteVoice` 之後、`#sound-upload` handler `:255` 之前）插入：

```js
  // ============== F14: orbit icon (pic) library state + rendering ==============
  let picAssignments = {}; // { waiting, completed, failed } -> filename | null (null = 光點)
  let picFiles = [];       // [{ name, size, mtime }]
  const PIC_MAX_BYTES_FE = 2 * 1024 * 1024;
  const PIC_EXT = ['png', 'jpg', 'jpeg', 'gif', 'webp', 'svg'];

  async function loadPicLib() {
    try {
      const res = await fetch('/api/pic/list');
      const data = await res.json();
      picFiles = data.files || [];
      picAssignments = data.assignments || {};
    } catch { picFiles = []; picAssignments = {}; }
    renderPicLib();
    renderPicAssigns();
  }

  function renderPicLib() {
    const lib = $('#pic-lib');
    if (!lib) return;
    if (!picFiles.length) { lib.innerHTML = '<div class="sound-lib-empty">（圖庫是空的，先上傳一張 icon）</div>'; return; }
    lib.innerHTML = '';
    for (const f of picFiles) {
      const row = document.createElement('div');
      row.className = 'sound-lib-row';
      row.innerHTML = `<img class="pic-thumb" alt="" src="/uploads/pic/${encodeURIComponent(f.name)}">`
        + `<span class="name" title="${escapeHtml(f.name)}">${escapeHtml(f.name)}</span>`
        + `<button class="pdel" title="刪除">🗑</button>`;
      row.querySelector('.pdel').onclick = () => deletePic(f.name);
      lib.appendChild(row);
    }
  }

  async function deletePic(name) {
    try {
      const res = await fetch('/api/pic/delete', {
        method: 'POST', headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ name }),
      });
      const data = await res.json();
      if (!res.ok) { pushToast({ title: '刪除失敗', msg: data.error || '' }); return; }
      picAssignments = data.assignments || {};
      await loadPicLib();
    } catch (err) { pushToast({ title: '刪除失敗', msg: err.message }); }
  }
```

在 voice 的事件指派區（`app.js:287-301` 的 `$$('.sound-assign')...` 區塊）之後、`// ============== Notification permission ==============`（`:303`）之前，插入：

```js
  // ============== F14: pic upload + event assignment ==============
  // 注意：#pic-upload / .pic-assign 的 markup 在 Task 4 加進 #sound-modal。
  document.addEventListener('change', async (e) => {
    if (e.target && e.target.id === 'pic-upload') {
      const file = e.target.files && e.target.files[0];
      e.target.value = '';
      if (!file) return;
      const ext = (file.name.split('.').pop() || '').toLowerCase();
      if (!PIC_EXT.includes(ext)) { pushToast({ title: '格式不支援', msg: '只接受 png / jpg / jpeg / gif / webp / svg' }); return; }
      if (file.size > PIC_MAX_BYTES_FE) { pushToast({ title: '檔案太大', msg: '單檔上限 2 MB' }); return; }
      try {
        const res = await fetch('/api/pic/upload?name=' + encodeURIComponent(file.name), {
          method: 'POST', body: file,
        });
        const data = await res.json();
        if (!res.ok) { pushToast({ title: '上傳失敗', msg: data.error || '' }); return; }
        await loadPicLib();
        pushToast({ title: '已上傳', msg: file.name });
      } catch (err) { pushToast({ title: '上傳失敗', msg: err.message }); }
    }
  });

  function renderPicAssigns() {
    $$('.pic-assign').forEach((sel) => {
      const event = sel.dataset.event;
      const current = picAssignments[event] || '';
      let html = '<option value="">（光點）</option>';
      for (const f of picFiles) {
        const selected = f.name === current ? ' selected' : '';
        html += `<option value="${escapeHtml(f.name)}"${selected}>${escapeHtml(f.name)}</option>`;
      }
      sel.innerHTML = html;
    });
  }

  document.addEventListener('change', async (e) => {
    const sel = e.target && e.target.closest && e.target.closest('.pic-assign');
    if (!sel) return;
    const event = sel.dataset.event;
    const name = sel.value || null;
    try {
      const res = await fetch('/api/pic/assign', {
        method: 'POST', headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ event, name }),
      });
      const data = await res.json();
      if (!res.ok) { pushToast({ title: '指派失敗', msg: data.error || '' }); return; }
      picAssignments = data.assignments || {};
    } catch (err) { pushToast({ title: '指派失敗', msg: err.message }); }
  });
```

> 為何用 delegated `document.addEventListener('change', …)` 而非 voice 的 `$$('.pic-assign').forEach` 直綁：voice 的 select 在 init 時 markup 已在 DOM，可直綁；pic 區同樣在 dashboard.html 靜態 markup（Task 4 加），其實也可直綁。**實作時若 Task 4 先完成**，可改成與 voice 完全一致的 `$$('.pic-assign').forEach(sel => sel.addEventListener('change', …))` 直綁版本（更貼合 F13 風格）。delegated 版是「markup 尚未存在時也安全」的保險寫法，二擇一皆可，擇一即可勿兩套並存。

**Files:**

> conic `::before`（`:318-339`）**完全不動**。icon 疊在其上。數值（base size / scale 曲線 / offset 起點）是視覺參數，**in-browser 微調**。

- [ ] **Step 1: 加 orbit-icon 定位 + 動畫**

在 `styles.css` 的 `@keyframes orbit-marquee { … }`（結束於 `:351`）**之後**插入：

```css
  /* F14: orbit icon — a real <img> riding the same perimeter as the conic light head.
     Visibility/animation are pure CSS cascade: only shows while the card needs-attention
     and the <img> isn't hidden (onerror sets hidden → falls back to the bare conic dot).
     Acknowledging attention removes .needs-attention → this selector stops matching →
     icon disappears automatically (no JS teardown needed). */
  .orbit-icon {
    position: absolute;
    top: 0;
    left: 0;
    width: var(--orbit-icon-base, 22px);
    height: var(--orbit-icon-base, 22px);
    object-fit: contain;
    pointer-events: none;
    z-index: 2;               /* above ::before (z-index:1) */
    display: none;            /* default hidden; shown only under needs-attention below */
    transition: opacity 0.15s ease-out;
  }
  .card.needs-attention .orbit-icon:not([hidden]) {
    display: block;
    offset-path: inset(-3px round calc(var(--radius) + 3px));
    offset-rotate: 0deg;      /* keep icon upright, don't rotate along the tangent */
    animation: orbit-icon-run 2.8s infinite;   /* same period as orbit-marquee */
  }
  /* Hover acknowledges attention — fade icon out with the glow (mirrors ::before) */
  .card.needs-attention:hover .orbit-icon { opacity: 0; }

  /* offset-distance %節點 + timing-function 與 orbit-marquee 完全對齊 → icon 在
     上/右/下/左 邊中點跟亮頭貼齊；scale 同節點變化做「上大下小」。
     數值（scale 0.5~1.0、base 22px、offset 起點）是視覺參數，in-browser 調到貼合即可。 */
  @keyframes orbit-icon-run {
    0%    { offset-distance: 0%;    transform: scale(1.0); animation-timing-function: ease-in; }  /* 上：最大 */
    40%   { offset-distance: 40%;   transform: scale(0.7); animation-timing-function: ease-in; }  /* 右：中 */
    61.5% { offset-distance: 61.5%; transform: scale(0.5); animation-timing-function: linear; }   /* 下：最小 */
    79%   { offset-distance: 79%;   transform: scale(0.7); animation-timing-function: ease-out; } /* 左：中 */
    100%  { offset-distance: 100%;  transform: scale(1.0); }                                       /* 回上 */
  }
```
